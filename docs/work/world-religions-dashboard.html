World Religions Dashboard

Maps global religious distribution, with particular methodological care on Norse/Neopagan Ásatrú data.

Like the Vampire Film Atlas, this was built as a gift — this time for a friend who practices Ásatrú, a modern Norse/Neopagan faith. I put real, documented care into getting that specific dataset right, more than a throwaway map would need, because it mattered to someone I know personally.

It maps global religious distribution more broadly, and connects naturally to my own ongoing interest in religion and Buddhist practice — not as an academic exercise, but as something I actually think about.

Live dashboard, global coverage.
